This project implements several algorithms to solve the {\bfseries{Knapsack problem}} — a classic optimization challenge. It includes\+:


\begin{DoxyItemize}
\item {\bfseries{Brute Force (Exhaustive Search)}}
\item {\bfseries{Dynamic Programming}}
\item {\bfseries{Greedy Algorithm}}
\item {\bfseries{Hybrid Algorithm}} (combines greedy with limited backtracking)
\end{DoxyItemize}

The program loads datasets describing pallets and truck capacities, runs selected algorithms, measures performance, and exports evaluation results.

\hypertarget{md__r_e_a_d_m_e_autotoc_md9}{}\doxysection{\texorpdfstring{Algorithms Summary}{Algorithms Summary}}\label{md__r_e_a_d_m_e_autotoc_md9}

\begin{DoxyItemize}
\item {\bfseries{Brute Force}}\+: Exhaustively tries all pallet combinations. Accurate but slow for large datasets.
\item {\bfseries{Dynamic Programming}}\+: Uses tabulation for efficient exact solution.
\item {\bfseries{Greedy}}\+: Picks pallets by highest profit-\/to-\/weight ratio, fast but approximate.
\item {\bfseries{Hybrid}}\+: Starts with greedy, extends candidate set, and applies limited backtracking for improvement.
\end{DoxyItemize}
